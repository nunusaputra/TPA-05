### **Daftar Endpoint :** <br> <br>

1. Membuat User Baru (Registrasi) :  `Method` POST `Endpoint` http://localhost:3000/skilvul/regis

2. Melakukan Login : `Method` POST `Endpoint`  http://localhost:3000/skilvul/login

3. Menambahkan Todo Baru : `Method` POST `Endpoint`  http://localhost:3000/skilvul/todos

4. Melihat Semua Todo (GET ALL TODO): `Method` GET `Endpoint`  http://localhost:3000/skilvul/todos

5. Melihat Todo (GET TODO) : `Method` GET `Endpoint`  http://localhost:3000/skilvul/todos/:id

6. Mengupdate Todo (UPDATE TODO) : `Method` PUT `Endpoint`  http://localhost:3000/skilvul/todos/:id

7. Menghapus Todo (DELETE TODO) : `Method` DELETE `Endpoint`  http://localhost:3000/skilvul/todos/:id

8. Menghapus Semua Todo (DELETE ALL TODO) : `Method` DELETE `Endpoint`  http://localhost:3000/skilvul/todos
